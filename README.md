# 🍰 Favorite Dessert

Welcome to **Favorite Dessert** — a fun and interactive dessert recipe app! You can discover, filter, and save your favorite desserts while practicing API integration, filtering, and responsive design.

This project is live on [Netlify](https://favorate-dessert.netlify.app/).

---

## 🚀 Features

- 🔍 **Search desserts** by name or favorite recipes
- 🍽️ **Cuisine filter**: Italy 🇮🇹, Asian 🌏, Middle East 🕌, or All 🌎
- 🔄 **Sorting options**: Descending ⬇️, Popularity ⭐, Random 🎲
- ❤️ **Like your favorite desserts** and find them in the **Favorites** section
- 📱 **Responsive design** — works beautifully on desktop 💻, tablet 📱, and mobile 📲

## 🧠 How It Works

1. The user can **search for a dessert** using the search bar.
2. Apply **filters by cuisine** or **sort recipes** by popularity, descending order, or random.
3. **Click the heart icon** to like a dessert; liked desserts appear in the **Favorites** section.
4. The app dynamically renders desserts with **images, cooking time, ingredients, and popularity**.
5. The layout is fully **responsive**, adapting to all screen sizes.
